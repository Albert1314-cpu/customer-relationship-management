## ❓ 常见问题

### Q: 如何添加新客户？
A: 在客户页面点击"+"按钮，填写客户信息后保存即可。

### Q: 如何导出客户数据？
A: 在设置页面选择"数据导出"，支持多种格式导出。

### Q: 数据存储在哪里？
A: 所有数据都存储在本地，确保您的隐私安全。

### Q: 支持哪些数据格式？
A: 支持 CSV、JSON 等常用格式的导入导出。
